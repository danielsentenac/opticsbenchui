INSTALLING OpticsBenchUI 1.1

External packages needed prior to installation:

Graphical user interface is based on Qt 5 or Qt 6 for Linux. Official Qt downloads: https://www.qt.io/download
Qt 5 source: git clone git://code.qt.io/qt/qt5.git
Qt 6 source: git clone git://code.qt.io/qt/qt6.git
OpticsBenchUI is compiled with Qt5 or Qt6 using the project file OpticsBenchUI.pro. First you need to update OpticsBenchUI.pro according to optional librairies installed and modify it with your personal path installations (INCLUDEPATH, LIBS tags)

OpticsBenchUI installation steps:

In the source directory (where is present the OpticsBenchUI.pro file):

1) qmake

  • Qt 5: qmake-qt5 OpticsBenchUI.pro
  • Qt 6: qmake6 OpticsBenchUI.pro

2) make

This will create an OpticsBenchUI executable

Optional dependencies (HDF5, OpenGL headers, Qt Multimedia, Standa, USB, Qt SVG) are detected at build time and disabled if missing.

3) Launch OpticsBenchUI. The documentation is accessible from the GUI and includes UI configuration guidance (Help -> User Manual / Getting Started). See the "Compilation Options" help page for a structured list of build defines.

Build options description:

Core data handling
HDF5 / NO_HDF5: Enable/disable HDF5 storage.
Cameras
USBCAMERA: Qt Multimedia camera backend.
NO_MULTIMEDIA: Disable Qt Multimedia camera backend.
IEEE1394CAMERA: libdc1394 + libraw1394 FireWire cameras.
NO_DC1394: Disable libdc1394 camera support.
NO_RAW1394: Disable libraw1394 integration.
GIGECAMERA: Aravis GenICam GigE/USB3 cameras.
NO_ARAVIS: Disable Aravis support.
NEOCAMERA: Andor Neo (SDK3 / atcore).
ZYLACAMERA: Andor Zyla (SDK3 / atcore).
NO_NEO: Disable Andor SDK3 cameras.
ALLIEDVISIONCAMERA: Allied Vision Vimba X SDK.
RASPICAMERA: Raspberry Pi camera (raspicam).
NO_RASPI: Disable raspicam support.
RAPTORFALCONCAMERA: Raptor Falcon (EPIX XCLIB).
RAPTORNINOX640CAMERA: Raptor Ninox640 (EPIX XCLIB).
NO_RAPTOR: Disable Raptor/EPIX cameras.
NO_USB: Disable USB comms (libusb).
Motion / motors
NO_STANDA: Disable Standa XIMC motors.
DAC / IO
ADVANTECHDAC: Advantech DAQNavi SDK.
NO_ADVDAQ: Disable Advantech DAC support.
RASPIDAC: Raspberry Pi SPI DAC (bcm2835).
NO_SPI: Disable SPI DAC support.
COMEDICOUNTER: Enable Comedi counter support (requires Qwt).
COMEDIDAC: Enable Comedi DAC support.
NO_COMEDI: Disable all Comedi support.
Notes
These flags are defined in OpticsBenchUI.pro based on detected SDKs.
If a dependency is missing, its feature is disabled automatically.

Convert between default and custom units.

Operation : ConvertUnit Performs the following conversion :

  • from custom unit to default unit
  • no conversion
  • from default unit to custom converts a value into default motion unit if it is expressed in custom one and gives the range associated to the unit
    Parameters
    unitmotion unit
    valueToConvertvalue to be converted
    rconvertedValueconversion result
    rrangeoutput range for the selected unit

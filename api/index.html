Optics bench control and data acquisition framework
OpticsBenchUI API

OpticsBenchUI is a Qt application for controlling optics-bench experiments and automated scans across cameras, motors, DACs, lasers, and counters.

Hardware families include USB/IEEE1394/GigE/Andor/Raptor/Raspberry Pi/Allied Vision cameras, Newport/New Focus/PI/Micos/Standa motors, NKT SuperK lasers, and Advantech/Comedi/Raspberry Pi DACs.

Group Overview:

  • Motors: Actuator drivers and motor control APIs.
  • Cameras: Camera interfaces and device implementations.
  • DAC/Comedi/Raspi/Advantech: Analog output devices and counters.
  • SLM: Spatial Light Modulators (currently acquired via HDMI snapshot).
  • Lasers: Laser sources and their drivers.
  • UI: Widgets and windows used in the GUI.
  • Acquisition: Acquisition workflow threads and helpers.
  • Analysis: Analysis workflow threads and helpers.
  • Utilities: Helpers, logging, and parsing.

File Configuration

The File functionality allows you to include data from external files in the same HDF5 dataset as OpticsBenchUI scans. This is useful when a device cannot be controlled directly but still produces files on disk.

File settings (Acquisition panel only)
DIR Directory path to watch for incoming files.
PREFIX File name prefix.
SUFFIX File extension (suffix).
DEL Delete files in DIR (and DIR/Results) at scan boundaries:
  • BEFORE
  • AFTER
SEP The data separator:
  • SPACE
  • COMMA
  • TAB
ORDER File selection order (FIFO-like):
  • LAST
  • FIRST
DELAY Delay (seconds) before grabbing the file.
FORMAT File format:
  • ASCII
TYPE Data type in the file. Examples:
  • ARRAY
  • LIST (optional label list: LIST[name1,name2,...])

OpticsBenchUI Documentation (version 1.1)

OpticsBenchUI controls devices commonly used in optical experiments. The exact hardware support depends on build options and installed drivers.

  • Comedi devices for acquisition and control (Configuration)
  • DAC boards to set voltages/currents (Configuration)
  • Supercontinuum sources (SuperK) (Configuration)
  • Motors to move optics (Configuration)
  • Cameras for beam profiling and imaging (Configuration)
  • SLMs to modulate phase and amplitude (Configuration)
  • File inputs for external data (Configuration)

Once devices are configured, scans are run from the Acquisition panel in the Operation menu (Configuration).

Data are saved in HDF5 files (accessible from C/Fortran and many other languages). See the HDF5 documentation.

SLM Configuration

An SLM (spatial light modulator) typically behaves like a display connected to the graphics card (HDMI/DisplayPort/DVI). Control consists of displaying an image in a dedicated SLM window, usually on an extended desktop.

Treat the SLM as a secondary monitor: set the SLM resolution/refresh rate in the OS display settings and place the SLM window on that screen. The software renders mask images full-screen in the SLM window.

SLM settings (Acquisition panel only)
IMAGEPATH Path prefix for the image location (first part of the full file path).
IMAGETYPE Image type / file extension.
START Starting image index when the filename includes a number.
STEP Increment step for image numbering (useful in loops).
SCREEN_X X coordinate of the top-left image origin on screen.
SCREEN_Y Y coordinate of the top-left image origin on screen.

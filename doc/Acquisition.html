Acquisition Configuration

The Acquisition window allows the user to program a Scan and save data in a user-specific way.

  • FILE: an external file is parsed to use its data
  • SLM: a spatial light modulator like a secondary screen is used with a registered masks list
    • Acquisition sequence
    record This is a unique value given by the user defining the sequence order in the scan. The scan will be treated in ascending order
    type The type of sequence. Can take one of the following name:
  • MOTOR: a motor is called for a movement
  • SUPERK: a supercontinnum laser is called for a filter setting
  • DAC: a DAC is called for setting an output value
  • CAMERA: a camera is called for taking an image
  • TREATMENT: a calculation on existing sequence data is performed
    		•
    instrument Here is indicated the instrument instance name from its configuration. Camera are denoted from 0 to n in order of appearance in the Instrument menu
    settings Here we put the information on what to do with the instrument, usually in format : [key=value] separated by one space
  • MOTOR: a motor can have MOVEREL (relative movement), MOVEABS (absolute movement) calls, with the value to be reached
  • SUPERK: a superk can have (R)POWER (power level), (R)ND (output level), (R)SWP (short wavelength setpoint), (R)LWP (long wavelength setpoint) calls with the value (absolute or (R) relative) to be reached
  • DAC: The DAC output is specified by OUTPUT key with the corresponding value to be set using key VALUE for absolute value or ke RVALUE for relative value
  • CAMERA: The SNAPSHOT key is used to grab an image from the camera to the format Mono 8-bit. The key SNAPSHOT32 is used to grab an image from the camera to the format Mono 12-bit, Mono 12-bit packed and Mono 16-bit
  • TREATMENT: The AVG=[i[-+/*]j] key is used to perform any type of average between two taken data sequences i and j (must be of the same instrument type). For example: AVG[10] will result on the average value taken from record 10.
    The PHASE=[i,j] key is used to compute the phase from imaginary i and real j parts.
    The AMPLITUDE=[i,j] key is used to compute the amplitude from imaginary i and real j parts.
    The IMAGE=[i] key is used to convert a 2D array dataset to the corresponding 8bit pixel image.
  • FILE: see File section.
  • SLM: see SLM section.
  • SLEEP: A sleep in seconds to be added at the end of the sequence
    • scanplan Here we describe the scan and closely related data saving behavior, usually in format : [key=value] separated by one space
  • LOOP: a fixed number of loop will begin when this key is present. It must be terminated with key LOOP=END. More than one loop can be nested
  • DATAGROUP: assign a name to a group of data. It may be used in relation with the LOOP key
  • DATANAME: assign a name to a data element. The data is attached to the parent group in the scanplan. If it is a scalar element, the data will be attached to the parent group as an attribute. If it is an averaged data (defined in the settings box), it will be attached to the grand parent group.

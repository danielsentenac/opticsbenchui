OpticsBenchUI Documentation

OpticsBenchUI allows to control 3 types of devices commonly used in optical experiments:

  • DACs used to control voltage/currents ( Configuration )
  • Motors to move mirrors and lens ( Configuration )
  • Cameras to look at beam profiles ( Configuration )

    • Once all devices are configured, a scan can be operated in the Acquisition panel from the Operation menu ( Configuration )

    Data are saved in HDF5 format in files. This format is fully accessible from C and Fortran code.
    A documentation can be found at HDF5 Reference Documentation

SUPERK Configuration

SUPERK devices are accessible from OpticsBenchUI through CP210X module for serial RS485 protocol. The type of SUPERK compatible devices are SuperK VARIA from NKT Photonics.

SUPERK settings
name This is a unique name given by the user defining the SuperK device instance
comtype The communication type (SERIAL_PORT)
comaddress The device address (/dev/ttyUSBXX)
comsettings The communication settings are for the serial port: baudrate,databits,parity,stopbit,vtime,flow control
description A brief description of the device

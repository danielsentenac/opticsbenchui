File Configuration

The File functionality allows users to store data from external files together with the data recorded by OpticsBenchUI scans. for example, it may happen when a device cannot be interfaced by OpticsBenchUI, but can store its data in files:

File settings (in the acquisition panel only)
DIR This is the directory path of the image location
PREFIX This is the file prefix
SUFFIX This is the file extension
DEL This indicates whether to delete the directory content

SLM Configuration

The SLM device (spatial light modulator) is in general a device that acts as a display monitor, and is connected to an output of the graphics card (VGA, DVI, HMI..). The control of the device consists of displaying an image onto the screen represented by the SLM window which may act as an extension of the primary monitor:

SLM settings (in the acquisition panel only)
IMAGEPATH This is the path of the image location. It shoudl be intended as the first part of a concatenated string indicating the full path name
IMAGETYPE This is the image type, or file extension
START This is the image starting number when it forms part of the image name
STEP This is the image incremental step number, when the sum START + STEP forms part of the image name. The STEP is useful to be used in a loop sequence
SCREEN_X This indicates where to display the x coordinate of the top/left image origin
SCREEN_Y This indicates where to display the y coordinate of the top/left image origin

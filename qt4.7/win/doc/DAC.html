DAC Configuration

DAC devices are accessible from OpticsBenchUI through PCI bus protocol. The type of PCI DAC compatible are from Advantech series PCI-1720U, PCI-1723, PCI-1724U, PCI-1727U.

Dac settings
name This is a unique name given by the user defining the DAC board instance
settings The communication address and some DAC parameters are specified like mode [CURRENT, VOLT], number of outputs, max and min value range. Here is an example of configuration setting line:
  • address=/dev/advdaq0 mode=CURRENT outputs=32 max=20 min=0
    		•
    description A brief description of the DAC board

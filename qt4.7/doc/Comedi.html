Comedi Configuration

Comedi devices are accessible from OpticsBenchUI using the comedi project. Refer to the documentation at comedi project web site.

Comedi settings
name This is a unique name given by the user defining the Comedi board instance
settings The file descriptor address and the mode of operation, the number of subdevices, the subdevice identifier range value (min to max). Here is an example of configuration setting line:
  • address=/dev/comedi0 mode=COUNTER subdevices=4 min=2 max=6
    		•
    description A brief description of the Comedi board
